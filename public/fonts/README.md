# Fonts

Place the display font **Aquavit (TT)** here so the site picks it up.

Expected filenames (referenced by `assets/css/main.css`):

- `Aquavit.woff2` (preferred)
- `Aquavit.woff`
- `Aquavit.ttf` (fallback)

You can convert a `.ttf`/`.otf` to web formats with e.g.
[Transfonter](https://transfonter.org/) or `woff2_compress`.

Until a file is present, the site falls back to a bold condensed system font.
